Detailed Description

A mixed type constant associated with a device

Instances of this class are only valid while the JabraConstants that delivered it is not yet disposed. In other words, a JabraConst belongs inside the using block with the JabraConstants. Don't save the JabraConst, but extract the values needed.

Operations on JabraConst objects are provided via extensions
