DeviceStatus


Exceptions
ExceptionThrown for any other exceptions.

Note: In order to connect to a device from the list of paired devices, make sure that there is no paired device currently connected. Any paired device currently connected has to be disconnected by calling DisconnectPairedDevice() before using ConnectPairedDevice. After device disconnection, GetPairedDevice() has to be called to get updated connection status.
