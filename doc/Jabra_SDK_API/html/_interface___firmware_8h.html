Enumeration Type Documentation

◆ DeviceFWURequirement

enum DeviceFWURequirement

This enum represents actions/parameters required to update firmware in a given device.

Enumerator
TunePackRegion 

It requires selecting a language region pack (display)

LanguageSelection 

It requires selecting a tune region pack

HasDockableHeadset 

It requires selecting a language (if a device requires region selection, it doesn't require language selection, and viceversa)

IsSelfPowered 

It has a dockable headset that must be docked before updating firmware

MustBePowerCycledAfterFwUpdate 

It is self-powered, therefore if it needs to power cycle it must be turned off and on

MustBeDockedInCradleAfterFwUpdate 

It must be power cycled after updating firmware

SupportsOtaUpdate 

It must be docked in the cradle after updating firmware
