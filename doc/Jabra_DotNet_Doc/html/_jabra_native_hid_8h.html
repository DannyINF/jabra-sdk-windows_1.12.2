◆ Jabra_HidState

enum Jabra_HidState

Enum for setting the HID working state.

Enumerator
NOT_SUPPORTED 

Setting Standard HID or GN HID is not supported

STD_HID 

Use standard HID as specified in the HID specification from usb.org.

GN_HID 

Use GN HID as specified by GN.
