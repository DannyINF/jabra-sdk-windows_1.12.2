const char *  deviceModel 
)

Get the URL for contacting Jabra customer support.

Parameters
[in]deviceIDID for specific device
[in]appNameName of the application
[in]appVersionVersion of the application
[in]deviceBrandName of the phone vendor (e.g., "Apple"). May be empty or NULL
[in]deviceModelDevice model name (e.g. "IPhone 8"). May be empty or NULL
Returns
