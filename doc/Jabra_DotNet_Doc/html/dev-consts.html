Device Constants

The device constants mechanism allows values specific to a device to be obtained from the SDK. The values are maintained by a cloud service, and will be specific to a device type and firmware version.

Each variable has a key, a unique string name, and a value of some type. The types are:

type default query access operation
int 0 Jabra_IsInt Jabra_AsInt
bool false Jabra_IsBool Jabra_AsBool
string "" Jabra_IsString Jabra_AsString
struct null Jabra_IsStruct Jabra_GetField
list null Jabra_IsList Jabra_ListElement, Jabra_AsInt for length

The default values will be returned if an access operation fails. Thus it is safe to chain access operations, any chain will just return the final default, if a link in the chain is broken.

To access the constants of a device, start with [Jabra_GetConstants](Jabra_GetConstants) method. The returned root pointer can then be used for getting individual variables by calling [Jabra_GetConst](Jabra_GetConst) with the root pointer and the key.

When the constants have been extracted the root handle must be released, by calling [Jabra_ReleaseConst](Jabra_ReleaseConst). This allows the SDK release the memory used for the constants, and does therefore invalidate all handles to constants obtained from that root.

For details on how the wrapper implements the interface, see wrapper.

Examples

In the following examples we access the value video-resolution which is a struct with two int fields named width and height

Using the values in C:

bool get_video_resolution(unsigned short deviceId, int* width, int* height) {
bool hasValue;
Jabra_Constants* root = Jabra_GetConstants(deviceId);
Jabra_Const resolution = Jabra_GetConst(root, "video-resolution");
has_value = Jabra_IsStruct(resolution);
*width = Jabra_AsInt(Jabra_GetField(resolution, "width"));
*height = Jabra_AsInt(Jabra_GetField(resolution, "height"));
Jabra_ReleaseConst(root);
return has_value;
}

Using the values in C#:

public Size GetResolution(IDevice device)
{
using (var jcc = device.GetConstants())
{
var c = jcc.GetConst("video-resolution");
return new Size { Width = c.GetField("width").AsInt(), Height = c.GetField("height").AsInt()};
}
}
