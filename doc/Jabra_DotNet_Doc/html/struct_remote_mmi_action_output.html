Detailed Description

Remote MMI action to use in Jabra_SetRemoteMmiAction for setting the MMI output LED(s). Only single bit value of RemoteMmiSequence can be specified as parameter RemoteMmiActionOutput to Jabra_SetRemoteMmiAction.

Note
RemoteMMIv2 only.
