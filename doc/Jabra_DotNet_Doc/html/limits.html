Obtaining parameter limits

Most of numeric values that can be set to control the video features share a common mechanism, the get limits operation. Invoking one of the get limits operations will give that range allowed for that value for the specific device.

Limits allows full control to the application programmer, by exposing the underlying integer values of the controls. For some of the values there is also a stepsize, representing the resolution of the value. If changes smaller than the stepsize is made they will still be saved, but will not affect any changes in the perceived value. The implicit stepsize for get limits operations without an explicit one is 1.

The get limits operations

  • Jabra_GetBrightnessLimits
  • Jabra_GetSharpnessLimits
  • Jabra_GetWhiteBalanceLimits
  • Jabra_GetSaturationLimits
  • Jabra_GetContrastLimits
  • Jabra_GetPanTiltLimits
  • Jabra_GetZoomLimits

Examples

public partial class Limits
{
public void SetRelativeZoom(int pct)
{
var limits = device.GetZoomLimits();
device.SetZoom((ushort) (pct * (limits.Max - limits.Min + 1) / 100));
}
}
