◆ IsFirmwareLockEnabled

bool JabraSDK.IFirmwareUpdate.IsFirmwareLockEnabled
get

Checks if firmware lock is enabled.

true if firmware lock is enabled; otherwise false.

Note: If the firmware lock is enabled, it is not possible to upgrade nor downgrade the firmware. In this situation the firmware can only be changed to the same version e.g. if you want to change the language.
