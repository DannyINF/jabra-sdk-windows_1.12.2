Device Constants

Access to the device constants (Core SDK function, Core info) is provided through a function on the device (JabraSDK::IDevice::GetConstants()). The returned value must be disposed, but can be used for obtaining individual constants. The access operations are for practical reasons implemented as extensions, but they can be seen in the below table.

Wrapper Core On
GetConst Jabra_GetConst Jabra_Constants
IsInt Jabra_IsInt Jabra_Const
IsBool Jabra_IsBool Jabra_Const
IsString Jabra_IsString Jabra_Const
IsStruct Jabra_IsStruct Jabra_Const
IsList Jabra_IsList Jabra_Const
AsInt Jabra_AsInt Jabra_Const
AsBool Jabra_AsBool Jabra_Const
AsString Jabra_AsString Jabra_Const
GetField Jabra_GetField Jabra_Const
ListElement Jabra_ListElement Jabra_Const
