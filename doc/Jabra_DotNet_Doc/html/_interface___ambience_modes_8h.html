◆ Jabra_GetAmbienceModeBalance()

LIBRARY_API Jabra_ReturnCode Jabra_GetAmbienceModeBalance ( unsigned short  deviceID,
Jabra_AmbienceMode  ambienceMode,
int8_t *  balance 
)

For a device supporting ambience modes, get the current right-left balance.

For a device supporting ambience modes, get the left-right balance of a given ambience mode:

  • 0 means neutral balance
  • negative values indicate balance on the left: the lower the number the more the balance is on the left.
  • positive values are for balance on the right: the higher the value, the more the balance is on the right. Returned values are constrained in a range [-N; N], where N is what Jabra_GetSupportedAmbienceModeBalance returns (e.g. if Jabra_GetSupportedAmbienceModeBalance returns 2, then the valid values are in the range [-2; 2]).
Parameters
[in]deviceIDID for the specific device
[in]ambienceModeThe mode of which we want to know the left-right balance of
[out]balanceThe variable filled with the left-right balance from the device. 0 is neutral balance, negative values are for balance on the left, positive numbers indicate balance on the right. The values are within a range [-N; N], where N is returned by Jabra_GetSupportedAmbienceModeBalance.
Returns
Return_Ok Call was successful
Device_Unknown deviceID is unknown
Not_Supported Functionality is not supported on this device
Return_ParameterFail balance is a NULL pointer or the mode doesn't support left-right balance (e.g. OFF).
Device_ReadFails Failed while reading from device
