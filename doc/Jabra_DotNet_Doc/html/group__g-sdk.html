LIBRARY_API bool Jabra_InitializeV2 ( void(*)(void)  FirstScanForDevicesDoneFunc,

bool  nonJabraDeviceDectection,

)

Library initialization - only intended to be called once.

Parameters
[in]FirstScanForDevicesDoneFuncCallback method, called when USB scan is done. Can be NULL if not used.
[in]DeviceAttachedFuncCallback method, called when a device is attached. Can be NULL if not used. Callee must call Jabra_FreeDeviceInfo to free memory.
[in]DeviceRemovedFuncCallback method, called when a device is removed. Can be NULL if not used.
[in]ButtonInDataRawHidFuncCallback method, called on new input data. HID Events will (default) NOT be triggered for standard HID to avoid conflicts Raw HID. Low-level. Can be NULL if not used.
[in]ButtonInDataTranslatedFuncCallback method, called on new input data. High-level. Can be NULL if not used.
[in]nonJabraDeviceDectectiontrue non Jabra and Jabra devices will be detected, false only Jabra devices will be detected. Non Jabra device detection is not supported on Linux
[in]configParamsOptional configuration of various SDK library behavior. Can be NULL if not used.
Returns
true if library initialization is successful.
false if library initialization is not successful. One reason could be that the library is already initialized or that Jabra_SetAppID has not been called prior to calling this function.
Note
AppID must be set using Jabra_SetAppID before the library initialization is called. If not the initialization fails.
