Whiteboard features

The Whiteboard features allow setup and selection of a video stream, intended to be used for a fixed area in the camera field of view that is occupied by a whiteboard.

Using the whiteboard consists of two parts:

  1. Selecting the field of view for the whiteboard. This is done by specifying the four corners of the part of the full camera view that the whiteboard occupy.

    The resolution of the full view is 3840 x 1080.

    Since the camera may not be directly facing the whiteboard, the corners need not form a rectangle. It will be transformed to a rectangle when the whiteboard stream is generated.

  2. Selecting where the whiteboard stream should be displayed. The whiteboard stream can be shown on the main video stream and on the secondary video stream, but not on both at the same time.

    Jabra_SetWhiteboardOnMainStream will disable or enable the whiteboard as the main stream. If the whiteboard is enabled multiple times it will enable the next available whiteboard on subsequent invocations.

    Invoking Jabra_SetSecondVideoStream with one of the Whiteboard values will set the whiteboard stream in the secondary stream.

Current devices support only one whiteboard, so all descriptions of handling multiple whiteboards are for future use.

Making a user interface for whiteboard

All the above set operations have get counterparts returning the current value of the setting. However, making a user interface especially for the whiteboard position may require some aditional elements.

The following steps might do a reasonable job:

  1. reset pan, tilt, and zoom to obtain the raw camera image
  2. show the stream (or a still picture from it) to the user
  3. show the current whiteboard corners on top of the picture
  4. let the user adjust the whiteboard corners on the picture
  5. apply the new corners as the whiteboard definition
  6. show the resulting whiteboard stream for user to judge the result

A simple way to reset the view to neutral is to use Jabra_RestoreVideoRoomDefaults.

The video stream can be obtained with the appropriate API for the operating system, see Video API.

Examples

public partial class Whiteboard
{
public static readonly int FullWidth = 3840;
public static readonly int FullHeight = 1080;
public void SetPosition(Size size, Point loLeft, Point loRight, Point upRight, Point upLeft)
{
device.SetWhiteboardPosition(0, new WhiteboardPosition()
{
LowerLeftCornerX = ScaleX(size, loLeft), LowerLeftCornerY = ScaleY(size, loLeft),
LowerRightCornerX = ScaleX(size, loRight), LowerRightCornerY = ScaleY(size, loRight),
UpperRightCornerX = ScaleX(size, upRight), UpperRightCornerY = ScaleY(size, upRight),
UpperLeftCornerX = ScaleX(size, upLeft), UpperLeftCornerY = ScaleY(size, upLeft),
});
}
private ushort ScaleX(Size size, Point p) { return (ushort) (FullWidth * p.X / size.Width); }
private ushort ScaleY(Size size, Point p) { return (ushort) (FullHeight * p.Y / size.Height); }
}
