.Net/C# Wrappers

The starting point for the .Net/C# wrappers are the IServiceFactory. This is where the application key is set, and the access point for different services.

Device Service

The device service is only the source for the IDevice instances, delivered by the device added event.

The device service is also the central registry for events for the devices.

See also
JabraSDK::IDeviceService

Integration Service

The integration service handles integration with other applications using the Jabra SDK (Jabra DIRECT, just to mention one). This is done by invoking [ConnectClient](JabraSDK::IIntegrationService::ConnectClient) on start and [DisconnectClient](JabraSDK::IIntegrationService::DisconnectClient) on end of the application.

See also
JabraSDK::IIntegrationService
