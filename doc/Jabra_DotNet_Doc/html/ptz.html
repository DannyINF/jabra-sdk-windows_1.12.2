Pan, tilt and zoom

Zooming is reducing the size of the view field, and to scale up that restricted view. Panning and tilting is moving the restricted view in the total field of view: Panning is moving the focus along the horizontal plane, and tilting is moving along the vertical plane. Even though these terms used to denote a physical change in a camera lens or mount, the cameras of the devices achieves the same effect by transforming the digital images.

Selecting pan, tilt, and zoom can be done either manually, or in an automatic mode controlled entirely by the device.

Video mode

The choice between manual and automatic video mode is controlled by the Jabra_SetVideoMode function.

  • FULL_SCREEN will select manual mode, where pan, tilt, and zoom can be set by the appropriate functions in the API
  • INTELLIGENT_ZOOM will let the device select the part of the view field where the action is.
  • ACTIVE_SPEAKER will use the device artificial intelligence to find the active speaker and then zoom there. The AI might not be perfect yet, but it tends to make meetings more entertaining.

In the future might be more available modes, and thus it becomes relevant to determine which modes are supported by a specific device. Since the set of available features depends not only on the device model but possible also on the firmware version, the SDK contains a query mechanism for answering just this question, see supported features.

Manual pan, tilt, and zoom

Zooming

The zooming can be set by the Jabra_SetZoom operation, and the current value read by the Jabra_GetZoom. A zoom value of 1 means not zoomed (full view), and larger values will represent a more zoomed view, that is a smaller view area. The maximum zoom supported can be obtained by Jabra_GetZoomLimits, see also the section on limits

Pan and tilt

Pan and tilt are set by the same command, Jabra_SetPanTilt, and can be read by Jabra_GetPanTilt. The unit for pan and tilt are degrees, positive pan will move the camera clockwise, and positive tilt will move the camera up. The ranges for pan and tilt can be obtained by the Jabra_GetPanTiltLimits operation, see also the section on limits.
