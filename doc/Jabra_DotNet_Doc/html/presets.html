Storing and applying preset value sets

To facilitate easy access to coherent sets of related settings the API supports two different preset value sets: Pan, tilt, and zoom preset (named PTZ preset), and the color control preset.

A preset works by having a number of banks, that will store a set of values. A store function will save the current values to the selected bank, and an apply function will apply all the values in a bank to the controls.

PTZ presets

The functions Jabra_StorePTZPreset and Jabra_ApplyPTZPreset will save and restore the pan, tilt, and zoom values. See pan, tilt, and zoom for further details.

<a name="presets-color"></a>Color control presets

The functions Jabra_StoreColorControlPreset and Jabra_ApplyColorControlPreset will save and restore brightness, sharpness, white balance, saturation, and contrast. See image settings for further details.
