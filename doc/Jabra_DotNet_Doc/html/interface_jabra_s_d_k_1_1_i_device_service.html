◆ PreloadDeviceInfo()

bool JabraSDK.IDeviceService.PreloadDeviceInfo ( string  zipFileName)

Preloads the configuration cache with the content of the specified archive.

Parameters
zipFileNameFull path name of the ZIP archive to preload from
Returns
true if preloading succeeds; otherwise false

Note: To get the full benefit, this should happen before calling IServiceFactory.CreateDeviceService(), as that enables device connections and may initiate background updates of device data. No existing data will be overwritten. This is deprecated. Please use the IDeviceCatalogue for IServiceFactory.CreateDeviceService().
