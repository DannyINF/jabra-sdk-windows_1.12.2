The Video Streams

Accessing the video streams from the Jabra devices is not a part of the SDK, but rather of the surrounding system. Hence integrating video feeds with your application is not directly related with the SDK operations. This allows you to use whatever library or system available in the context of your application to show the feed.

When a device is detected by the SDK, the name of the device will match with the name of the video feed. So an application may choose to search among the available video streams for a matching stream.

The following example shows an integrated video stream in a .NET C# application, using WPFMediaKit and DirectShow.

Example

The C# class used for showing a video stream, defaulting to the stream with the same name as the device just added:

public class VideoViewModel : ViewModel
{
private int _deviceIndex;
private object _videoDevice;
public VideoViewModel(IDevice device)
{
VideoDevices = new ObservableCollection<string>(MultimediaUtil.VideoInputNames);
DeviceIndex = Array.IndexOf(MultimediaUtil.VideoInputNames, device.Name);
}
public ObservableCollection<string> VideoDevices { get; }
public int DeviceIndex
{
get => _deviceIndex;
set
{
if (_deviceIndex == value) return;
_deviceIndex = value;
VideoDevice = value != -1 ? MultimediaUtil.VideoInputDevices[value] : null;
}
}
public Object VideoDevice
{
get => _videoDevice;
private set { _videoDevice = value; RaisePropertyChanged(); }
}
}

The xaml snippet with the video also contains a list of all available Video streams, and allows selecting an alternative stream, mainly to allow a view of alternative streams from the device:

<DataTemplate DataType="{x:Type vm:VideoViewModel}">
<Grid>
<Grid.ColumnDefinitions>
<ColumnDefinition Width="Auto"/>
<ColumnDefinition Width="*"/>
</Grid.ColumnDefinitions>
<Grid.RowDefinitions>
<RowDefinition Height="*"/>
</Grid.RowDefinitions>
<ListBox Grid.Row="0" Grid.Column="0" ItemsSource="{Binding VideoDevices}" SelectedIndex="{Binding DeviceIndex}" />
<controls:VideoCaptureElement Grid.Row="0" Grid.Column="1" VideoCaptureDevice="{Binding VideoDevice}"/>
</Grid>
</DataTemplate>
