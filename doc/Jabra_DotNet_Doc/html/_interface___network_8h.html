◆ Jabra_ConfigureXpressManagement()

LIBRARY_API Jabra_ReturnCode Jabra_ConfigureXpressManagement ( unsigned short  deviceID,
const char *  xpressurl,
const ProxySettings *  proxy,
unsigned int  timeout 
)

Configures Xpress management related settings on a network-capable device. Will enable Ethernet interface if no network interfaces are available. In case of NetworkRequest_Fail or Return_Timeout return codes, the proxy configuration and Xpress URL will revert to the previous values.

Parameters
[in]deviceIDID for a specific device
[in]xpressurlPointer to NULL-terminated string containing the URL for the Xpress management backend. Protocol is always assumed to be HTTPS no matter the prefix of the URL. Setting URL pointer to NULL will disable management.
[in]proxyPointer to proxy settings - set to NULL if not using proxy (default)
[in]timeoutMaximum allowed execution time for the entire operation. In the worst cast the flow is:
  1. Enabling network interface
  2. Waiting for DHCP resolution
  3. Startup of management service
  4. DNS resolution of specified URL
  5. HTTPS request and parsing the response The recommended value is 30'000 ms. Setting timeout=0 will configure the device and return immediately without checking the validity of the xpressurl.
Returns
Return_Ok Device was successfully configured for stand-alone remote management
Device_Unknown deviceID is unknown
Return_ParameterFail A parameter was missing or incorrect
Not_Supported Functionality is not supported on this device
Device_WriteFail Failed while writing to device
NetworkRequest_Fail URL was unreachable - see Jabra_GetXpressManagementNetworkStatus
Return_Timeout URL validation did not complete before timeout (when timeout>0)
See also
Jabra_GetXpressManagementNetworkStatus

◆ Jabra_EnableRemoteManagement()

LIBRARY_API Jabra_ReturnCode Jabra_EnableRemoteManagement ( unsigned short  deviceID,
bool  enable,
unsigned int  timeout 
)

Enables or disables remote management. If enabling and there are no active network interfaces (eth/wlan), it will automatically enable the Ethernet interface as well since remote mgmnt requires a network. If a timeout is specified, it will wait for an interface to be IP-configured before returning.

Parameters
[in]deviceIDID for a specific device
[in]enableEnable state: true=enabled, false=disabled
[in]timeoutTime to wait for ethernet interface to become ready. Set to 0 to return immediately.
Returns
Return_Ok Remote management is enabled
Device_Unknown deviceID is unknown
Not_Supported Functionality is not supported on this device
Device_WriteFail Failed while writing to device
NetworkRequest_Fail Management was enabled, but no working network interface was available at the time of return.
See also
Jabra_GetEthernetIPv4Status

[out]errorcodePointer to int that will receive libcurl error code. If NULL, no error code will be returned.
