Image settings

This is the values for controling details of the video/image settings. They all support both get and set, as well as a get limits operation that will obtain the allowed range for the value. All the image settings are stored and retrieved when the color control presets are used.

The list of settings:

Get: Set:
Jabra_GetContrastLevel Jabra_SetContrastLevel
Jabra_GetSharpness Jabra_SetSharpness
Jabra_GetWhiteBalance Jabra_SetWhiteBalance
Jabra_GetSaturation Jabra_SetSaturation
Jabra_GetBrightness Jabra_SetBrightness
